# One Night

2DアクションRPG（Unity / C#）個人制作

## 概要
オリジナルのイラスト素材を使用した2DアクションRPGです。
拡張性を意識した設計で、基盤設計から実装まですべて一人で制作しました。
現在も継続的にブラッシュアップを行っています。

## 使用技術
- Unity
- C#
- Procreate（オリジナルイラスト素材制作）
- Shader Graph（風で揺れる草の表現）
- Cinemachine（カメラ演出）

## 主な実装内容
- FSMによるプレイヤー／敵AIの状態管理
- AnimatorのBlend Treeを用いた移動アニメーション制御
- アニメーションイベントによる攻撃判定制御
- ScriptableObjectによるアイテム・ステータスデータ管理
- コヨーテタイム、ジャンプバッファなどの操作感改善
- 回復時カメラ振動（Cinemachine Impulse）

## 操作方法
| アクション | キーボード | ゲームパッド |
|---|---|---|
| 移動 | A / D | 左スティック |
| ジャンプ | Space | A / × |
| ダッシュ | J | B / ○ |
| 攻撃 | K（連打でコンボ） | X / □ |
| 遠距離攻撃 | L | 左ショルダー |
| 回復 | C（長押し） | 右スティック押し込み |

約3ヶ月（個人制作）
